Guía de Usuario - Módulo de Mantenimiento

Capa Modelo - Funcionalidades de Mantenimiento

El módulo de mantenimiento permite gestionar los registros de mantenimientos realizados a las maquinarias en la base de datos de la empresa. Este módulo se conecta con la base de datos para registrar, actualizar y desactivar los mantenimientos según sea necesario.

Capa Control - Funciones Principales

La capa de control sirve como intermediaria entre la vista del usuario y la base de datos. Aquí se gestionan las solicitudes realizadas por el usuario y se envían al modelo para su procesamiento.

Agregar Mantenimiento

Permite registrar un nuevo mantenimiento para una maquinaria en el sistema. Los datos incluyen el nombre de la maquinaria, tipo, horas de operación, mantenimiento periódico, entre otros detalles.

Actualizar Mantenimiento

Actualiza un registro de mantenimiento existente con la información más reciente. Ideal para modificar detalles como el próximo mantenimiento o el área donde se utiliza la maquinaria.

Desactivar Mantenimiento

Desactiva un mantenimiento, marcándolo como inactivo. Esto conserva el historial sin eliminar información relevante.

Consultar Mantenimientos

Permite ver todos los mantenimientos registrados en el sistema, incluidos los detalles importantes para cada maquinaria.

Obtener Último ID

Consulta el último ID de maquinaria registrado para facilitar el ingreso de un nuevo mantenimiento sin duplicar IDs.

Capa Vista - Formulario de Mantenimiento

El formulario Frm_Mantenimiento_Produccion es la interfaz de usuario para el módulo de mantenimiento. Desde aquí se pueden agregar, actualizar, desactivar y consultar mantenimientos.

Opciones del Formulario

Botón "Agregar"

Permite agregar un nuevo mantenimiento. Completa todos los campos requeridos y presiona "Agregar" para guardarlo en el sistema.

Botón "Actualizar"

Actualiza el mantenimiento seleccionado en el sistema. Selecciona un registro, modifica los datos necesarios y presiona "Actualizar".

Botón "Desactivar"

Desactiva el mantenimiento seleccionado. Marca el registro como inactivo sin eliminarlo, conservando el historial.

DataGridView - Lista de Mantenimientos

Muestra todos los mantenimientos en el sistema. Haz clic en un registro para visualizar y modificar los detalles en el formulario.

Campo "Último ID de Maquinaria"

Muestra el último ID registrado y sugiere un nuevo ID para el próximo mantenimiento.
